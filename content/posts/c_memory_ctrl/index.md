c 语言使用指针来进行内存管理, 指针具备类型的概念, 这个类型指的其实是它指向变量的类型, 该类型决定了它占据了多大一片内存. 然后事实上, 所有的指针, 不论其类型如何, 在 c 语言中都被统一表示为一个整数.

这其中存在很大的问题: 指针其实并包含任何指示大小的信息, 它被打印出来只是一个整数而已. 
那么如 malloc, free 的方法是如何准确地为指针指向的变量分配/清除内存的呢.
